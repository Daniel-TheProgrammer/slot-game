### How it works

1. You won!: (3 symbols in the middle horizontal row)
   - 3 Same symbols
   - 2 Same symbols and 1 Wild
   - 2 Wilds and 1 any symbol
   - _**You should note that:**_ 3 Wilds counts as a loss
